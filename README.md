# Resonant

A simple resonant breathing timer with gentle chimes.

## Features

- **Visual breathing guide** - Circle expands/contracts with your breath
- **Audio cues** - Gentle chimes mark each phase transition
- **Customizable timing** - Adjust breath duration (2-8s per phase)
- **Session timer** - Set your meditation length (1-20 minutes)
- **Box breathing pattern** - Inhale → Hold → Exhale → Hold

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS v4
- shadcn/ui
- Web Audio API (for chimes)

## Breathing Patterns

Default is box breathing (equal phases). Resonant breathing typically uses 4-6 second phases for optimal heart rate variability benefits.
